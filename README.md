# GoogleDocs-clone

Google Docs clone for CSE 356

### How to RUN

#### In Local:

- `yarn workspace (client || server) start` in the very highest folder.

#### In Cloud:

- add content

### Possible Errors

1. Module not installed
   - move to client or server folder
   - run `npm install {module}@{version}`
     // `yarn install {module}@{version}` doesn't work somehow...
